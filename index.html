2012 年

03 月 26 日 ( 月 )

ええわぁ。

実際にメンバーの誰かの家に集合ってあるんかな。他所のことはわからんのだけれど。

自分がいたバンドのメンバーは、自分を含めてもともと大阪の北摂地域に住んでいたのだけれど、ぼちぼちみんな結婚しはじめて北は滋賀県守山市、南は大阪府堺市と住んでるところがバラつくようになった。なのでリハスタだろうがライブハウスだろうがいつも現地集合だった。

そういえば他のメンバーとはリハスタやライブハウス以外で顔を合わせることがまったくなかった。そのためかリハをやろうがライブをやろうが、なんだかまるで仕事みたいって感じてた。でも、そういう音楽だけでつながってるようなメンバーとのつきあい方が自分はけっこう好きだった。

まぁ、ある日、ライブ前にウイスキーのボトルを空けてベロンベロンになって、まともに演奏できなかった Bass と Guitar の二人には当時は正直ムカついたけど ( 笑 ) 飲んでる理由はわかってたけど、それでもやっぱりムカついた ( 笑 )

あ゛。そういえば Bass と Guitar の結婚式で演奏してるの思い出した。やっぱり現地集合だけど。

Bass の結婚式の二次会は京都のライブハウスの小ホールを借り切ってなんか規模がやばかった。200 人くらい入りそうなハコが人で埋まってたもんなぁ。まぁ結婚式の二次会だから埋まってたわけだけど。ふだんのライブは閑散としてる ( 笑 )

そういえば Guitar の披露宴 ( 二次会じゃないよ！！ ) で演奏したときは、当然披露宴会場にドラムセットがなくて、ドラムセットをレンタルして持ち込んだんだった。持ち込んだドラムセットを見て会場の支配人が固まってた ( 笑 ) 演奏するとは聞いていたけど、セットの持ち込みは予想外のできごとだったんだろうね ( 笑 )

あ、自分の結婚式の二次会での演奏はノーカンでよろしく ( 笑 )

なんだか音楽をやる話が全部過去形になっちゃうね。仕方ないけどね。
